# Wireless PC Power

Control your PC Power in any device

### Few things that you might need to change in the code

| Line                                                         | Changes                                                     |
| ------------------------------------------------------------ | ----------------------------------------------------------- |
| `const char ssid[] = "";`                                 | Put in your WIFI SSID here                                  |
| `const char pass[] = "";`                                 | Put in your WIFI password here                              |
